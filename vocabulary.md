<h2>Basic Verbs</h2>
<p>Here is a table showing you how to use the very common verb <b><i>être</i></b>, which means 'to be'</p>
<table style="width:100%">
  <tr>
    <th>French</th>
    <th>English</th> 
  </tr>
  <tr>
    <td>Je suis</td>
    <td>I am</td> 
  </tr>
  <tr>
    <td>Tu es</td>
    <td>You are</td> 
  </tr>
   <tr>
    <td>Il/elle/on est</td>
    <td>He/she/it is</td> 
  </tr>
  <tr>
    <td>Nous sommes</td>
    <td>We are</td> 
  </tr>
  <tr>
    <td>Vous etes</td>
    <td>You (pl) are</td> 
  </tr>
  <tr>
    <td>Ils/elles sont</td>
    <td>They are</td> 
  </tr>
</table>
